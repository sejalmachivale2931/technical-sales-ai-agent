# Technical Sales Engineer AI Agent

## Overview
This project demonstrates an AI-inspired Technical Sales Engineer Agent.

The agent reads a client meeting transcript and automatically generates a Solution Design Document.

The output includes:
- Pain Points
- Current Technology Stack
- Desired Outcome
- Suggested Solution Architecture
- Implementation Steps

## Files

agent.py → main agent logic  
meeting1.txt → sample meeting transcript  
meeting2.txt → second transcript for testing dynamic behavior  

## How to Run

1. Download the project
2. Open project folder
3. Run the command:

python agent.py

4. Output file will be generated:

solution_design.txt

## Example Use Case

Input: client meeting discussion  
Output: structured technical solution document

This demonstrates an end-to-end workflow for AI-assisted technical sales solution design.
